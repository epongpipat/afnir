# afnir: Wrapper Functions for 'AFNI' (Analysis of Functional 'NeuroImages')

The goal of afnir is to wrap AFNI functions in R.

## Installation

You can install afnir from github with:

```R
devtools::install_github("epongpipat/afnir")
```
